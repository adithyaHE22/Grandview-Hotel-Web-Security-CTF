For a complete experience, add the following images:

- `room101.jpg` - Single room with city view
- `room102.jpg` - Double room with garden view

### Additional Assets

## Image Specifications
- **Format:** JPEG or PNG
- **Resolution:** 800x600 pixels minimum
- **Size:** Under 1MB per image for web optimization
- **Aspect Ratio:** 4:3 recommended for consistency

## Note
The application will work without images - placeholder images will be used automatically if files are missing.
